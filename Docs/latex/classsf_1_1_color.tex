\doxysubsection{Detailed Description}
Utility class for manipulating RGBA colors. 

\mbox{\hyperlink{classsf_1_1_color}{sf\+::\+Color}} is a simple color class composed of 4 components\+:

\begin{DoxyItemize}
\item Red \item Green \item Blue \item Alpha (opacity)\end{DoxyItemize}
Each component is a public member, an unsigned integer in the range \mbox{[}0, 255\mbox{]}. Thus, colors can be constructed and manipulated very easily\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classsf_1_1_color}{sf::Color}} color(255, 0, 0); \textcolor{comment}{// red}}
\DoxyCodeLine{color.r = 0;                \textcolor{comment}{// make it black}}
\DoxyCodeLine{color.b = 128;              \textcolor{comment}{// make it dark blue}}

\end{DoxyCode}


The fourth component of colors, named \char`\"{}alpha\char`\"{}, represents the opacity of the color. \mbox{\hyperlink{class_a}{A}} color with an alpha value of 255 will be fully opaque, while an alpha value of 0 will make a color fully transparent, whatever the value of the other components is.
