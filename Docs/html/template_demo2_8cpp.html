Detailed Description

Class Template Demo.

Class Templates like function templates, class templates are useful when a class defines something that is independent of the data type. Can be useful for classes like LinkedList, BinaryTree, Stack, Queue, Array, etc.

template < typename T > template < class T >

are equivalent. Having said that, there are specific cases where there is a difference between typename and class. The first one is in the case of dependent types. typename is used to declare when you are referencing a nested type that depends on another template parameter, such as the typedef in this example:

template<typename param_t> class Foo { typedef typename param_t::baz sub_t; };
